bat-toolkit

Obfuscate & restore .bat files

A simple browser-based toolkit for injecting and stripping obfuscation prefixes. No install, no upload — runs entirely client-side.

Obfuscator
Injects an 8-byte prefix into your .bat file
Encode
Drop your .bat file or click to browse Outputs as filename_obfuscated.bat
Payload (base64)
//4mY2xzDQo=
Decoded (hex)
FF FE 26 63 6C 73 0D 0A
Prefix content
BOM + &cls\r\n
Prefix size
8 bytes
Deobfuscator
Strips the injected prefix and restores original
Decode
Drop your obfuscated .bat file or click to browse Outputs as filename_deobfuscated.bat
⚠
Compatibility notice Only works with files using the prefix injection method (//4mY2xzDQo=). Files from other obfuscators will not be recognized.
