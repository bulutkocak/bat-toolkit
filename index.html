bat-toolkit

Obfuscate & restore .bat files

Four obfuscation methods, all fully reversible. No install, no upload — runs entirely client-side.

Obfuscator
Select method & encode
Pick a method, then drop your .bat file
Encode
Prepends an 8-byte payload (BOM + &cls) before the file content.
Drop your .bat file or click to browse Outputs as filename_obfuscated.bat
Method reference
How each obfuscation works
METHOD 1 — PREFIX INJECTION
Prepends FF FE 26 63 6C 73 0D 0A (BOM + &cls\r\n) to the raw file bytes.
METHOD 2 — BASE64
Encodes the entire file as base64 and wraps it in a certutil self-decoding stub.
METHOD 3 — JUNK LINES
Inserts random REM comments and blank lines between every real line of the script.
METHOD 4 — VAR SCRAMBLE
Replaces all %variable% names with random tokens. Map is embedded in the file header.
Deobfuscator
Select method & decode
Must match the method used to obfuscate
Decode
Detects and strips the 8-byte prefix, restoring original file bytes.
Drop your obfuscated .bat file or click to browse Outputs as filename_deobfuscated.bat
⚠
Compatibility notice The selected decode method must match the one used to obfuscate the file. Mismatched methods will fail or produce corrupted output.
